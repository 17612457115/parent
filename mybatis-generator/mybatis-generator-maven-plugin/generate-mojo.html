MyBatis Generator Maven Plugin

  • Last Published: 24 June 2016 |
  • Version: 1.3.3

mybatis-generator:generate

Full name:

org.mybatis.generator:mybatis-generator-maven-plugin:1.3.3:generate

Description:

Goal which generates MyBatis/iBATIS artifacts.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: generate-sources.

Required Parameters

Name Type Since Description
configurationFile File - Location of the configuration file.
Default value is: ${project.basedir}/src/main/resources/generatorConfig.xml.
User property is: mybatis.generator.configurationFile.
outputDirectory File - Output Directory.
Default value is: ${project.build.directory}/generated-sources/mybatis-generator.
User property is: mybatis.generator.outputDirectory.

Optional Parameters

Name Type Since Description
contexts String - Comma delimited list of contexts to generate.
User property is: mybatis.generator.contexts.
jdbcDriver String - JDBC Driver to use if a sql.script.file is specified.
User property is: mybatis.generator.jdbcDriver.
jdbcPassword String - JDBC password to use if a sql.script.file is specified.
User property is: mybatis.generator.jdbcPassword.
jdbcURL String - JDBC URL to use if a sql.script.file is specified.
User property is: mybatis.generator.jdbcURL.
jdbcUserId String - JDBC user ID to use if a sql.script.file is specified.
User property is: mybatis.generator.jdbcUserId.
overwrite boolean - Specifies whether the mojo overwrites existing files. Default is false.
Default value is: false.
User property is: mybatis.generator.overwrite.
skip boolean - Skip generator.
Default value is: false.
User property is: mybatis.generator.skip.
sqlScript String - Location of a SQL script file to run before generating code. If null, then no script will be run. If not null, then jdbcDriver, jdbcURL must be supplied also, and jdbcUserId and jdbcPassword may be supplied.
User property is: mybatis.generator.sqlScript.
tableNames String - Comma delimited list of table names to generate.
User property is: mybatis.generator.tableNames.
verbose boolean - Specifies whether the mojo writes progress messages to the log.
Default value is: false.
User property is: mybatis.generator.verbose.
