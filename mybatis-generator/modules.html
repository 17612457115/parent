Project Modules

This project has declared the following modules:

Name Description
MyBatis Generator Core MyBatis Generator - a code generator for MyBatis and iBatis.
MyBatis Generator Tests (Java DOM Tests) Tests for the Java DOM classes. Generates Java classes and fails if the classes won't compile.
MyBatis Generator Maven Plugin Maven plugin for MyBatis Generator.
MyBatis Generator Tests (Common Classes) Shared classes for all the test project.
MyBatis Generator Tests (MyBatis3) Tests for the MyBatis3 code generator.
MyBatis Generator Tests (iBATIS2 - Java2) Tests for the iBatis code generator in a Java 2 environment (no generics)
MyBatis Generator Tests (iBATIS2 - Java5) Tests for the iBatis code generator in a Java 5 environment (with generics)
